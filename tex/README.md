# LaTeX

I love LaTeX. It's far superior to Word in its output although the learning
curve is ferocious.

## Generating PDF's.

  1. You need LaTeX for Windows, Mac or Linux. It's big: 3GB+
  2. The command to generate the PDF is: `pdflatex mmai831_watts_aios1.tex`
